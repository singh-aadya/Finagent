Two-Agent Architecture

Two Agents. One Goal.

Each agent is a specialised GPT-4o instance with a distinct system prompt, clear role, and scoped responsibility.

01
Agent 01

Data Agent

Fetches and structures all raw financial data. It downloads price history, computes key ratios, calculates correlations, and builds a clean data summary — ready for analysis.

  • Historical price data via yfinance
  • P/E, Forward P/E, ROE, Debt/Equity
  • Normalised price chart generation
  • Correlation coefficient calculation
  • % performance over chosen period
System Role "You are a financial data analyst. Your job is to process the raw data provided and return a structured JSON summary ready for report writing. Be precise and factual."
Passes structured data
02
Agent 02

Report Agent

Receives the structured data summary from Agent 01 and writes a complete, professional equity research report in Markdown — with analysis, comparisons, and a future outlook.

  • Executive summary & key takeaways
  • Metrics comparison table
  • Ratio explanations in plain English
  • Correlation & risk analysis
  • Future scenario analysis
System Role "You are a senior equity research analyst. Write a professional financial report in Markdown using only the data provided. Include a table, metric explanations, and a future outlook."
Flow

How It Works

1

Enter Your Inputs

Open the Streamlit app in your browser. Enter two stock tickers, your OpenAI API key, and a time period.

2

Data Agent Activates

Agent 01 fetches live price data and financial metrics from Yahoo Finance. It computes correlations, normalises prices, and assembles a structured JSON data package.

3

Report Agent Writes

Agent 02 receives the data package and writes a full institutional-grade equity report with tables, analysis, and outlook — in seconds.

4

Download & Share

The report and chart appear instantly in the UI. Download as .md and .png files with one click.

Quick Start

Run in 3 Steps

Step 1 — Install
terminal 
# Clone or unzip the project
cd finagent

# Install all dependencies
pip install -r requirements.txt
Step 2 — Launch
terminal 
# Start the Streamlit app
streamlit run app.py

# Opens at http://localhost:8501
Step 3 — Analyse
browser 
# In the sidebar:
API Key  → your OpenAI key
Stock 1  → AMD
Stock 2  → NVDA
Period   → 6mo

# Click: 🚀 Generate Report
